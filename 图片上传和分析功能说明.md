# 小红书采集工具 - 图片上传和分析功能

## 新增功能

### 1. 图片上传功能
在编辑预览界面中，用户可以手动上传图片：

- **上传按钮**：在图片选择区域右上角添加"上传图片"按钮
- **多文件支持**：支持一次选择多个图片文件
- **文件类型**：接受所有图片格式（image/*）
- **即时显示**：上传的图片立即显示在图片网格中
- **自动选择**：新上传的图片默认被选中

### 2. 图片管理功能
增强的图片管理功能：

- **删除功能**：每张图片左上角添加删除按钮
- **选择/取消**：点击图片或复选框来选择/取消选择
- **下载功能**：保留原有的单张图片下载功能
- **状态显示**：实时显示已选择的图片数量

### 3. 图片分析功能
上传图片时自动分析产品类型：

- **智能分析**：基于图片内容自动识别产品类型
- **API集成**：调用 `/api/analyze-image` 接口进行分析
- **类型更新**：分析结果自动更新产品类型字段
- **置信度显示**：显示分析的置信度（60-90%）

```typescript
// 图片上传事件处理
onChange={async (e) => {
  const files = e.target.files
  if (files && files.length > 0) {

    })

  }
}}
```

#### 图片分析API
```typescript
// 图片分析API调用
async function analyzeImageWithAPI(imageFile: File, currentProductType: string): Promise<string> {
  try {
    // 将图片转换为base64
    const base64 = await new Promise<string>((resolve, reject) => {
      const reader = new FileReader()
      reader.onload = () => resolve(reader.result as string)
      reader.onerror = reject
      reader.readAsDataURL(imageFile)
    })
    
    const response = await fetch('/api/analyze-image', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        image: base64,
        currentProductType: currentProductType
      })
    })
    
    const data = await response.json()
    
    if (data.success) {
      console.log('图片分析结果:', data.data)
      return data.data.productType
    } else {
      console.warn('图片分析失败:', data.error)
      return currentProductType
    }
  } catch (error) {
    console.error('图片分析API调用失败:', error)
    return currentProductType
  }
}
```

## 使用方法

### 1. 上传图片
1. 点击编辑预览界面中的"上传图片"按钮
2. 选择要上传的图片文件（支持多选）
3. 图片会自动显示在图片网格中
4. 系统会自动分析第一张图片并更新产品类型

### 2. 管理图片
1. **选择图片**：点击图片或复选框来选择/取消选择
2. **删除图片**：点击图片左上角的删除按钮
3. **下载图片**：点击图片右下角的下载按钮
4. **查看数量**：底部显示已选择的图片数量

### 3. 产品类型分析
1. 上传图片时，系统会自动分析第一张图片
2. 分析结果会更新产品类型字段
3. 如果当前产品类型是"未分类"，一定会使用分析结果
4. 如果分析置信度很高（>80%），也会更新产品类型

## 注意事项

1. **文件大小限制**：建议限制上传图片的大小（如单张<10MB）
2. **并发处理**：大量图片上传时考虑分批处理
3. **内存管理**：及时清理不再使用的图片URL对象
4. **隐私保护**：上传的图片仅用于分析，不会永久存储

这个功能让用户可以完全手动控制图片内容，同时通过AI分析获得智能化的产品类型建议。
