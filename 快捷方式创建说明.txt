小红书采集工具 - 桌面快捷方式创建说明

由于自动创建快捷方式遇到编码问题，请按照以下步骤手动创建：

方法1：手动创建快捷方式
1. 在桌面上右键点击空白处
2. 选择"新建" -> "快捷方式"
3. 在"请键入对象的位置"中输入：
   cmd.exe /k "cd /d "G:\2.研究生\1.研一文件\6.Claude\redbook" && npm run dev"
4. 点击"下一步"
5. 在"键入该快捷方式的名称"中输入：
   小红书采集工具
6. 点击"完成"

方法2：使用现有启动脚本
1. 双击运行项目文件夹中的 start-app.cmd
2. 应用会自动启动并打开浏览器

方法3：直接访问
1. 确保开发服务器正在运行（npm run dev）
2. 在浏览器中访问：http://localhost:3001

注意事项：
- 如果端口3001被占用，应用会自动切换到其他端口
- 确保Node.js和npm已正确安装
- 项目依赖已安装（node_modules文件夹存在）

快捷方式创建完成后，双击即可启动应用。
